## 🚀 Quick Start (4 Simple Steps!)

### Step 1: Fork & Clone

```bash
# 1. Fork on GitHub first
# 2. Then clone:
git clone https://github.com/YOUR_USERNAME/pivalue.world.git
cd pivalue.world
```

### Step 2: Run Calculation

```bash
python src/piclalculation.py
```

- Enter GitHub username
- Choose time limit (2/5/10 min)
- Get Verification Code + Submission ID
- **Auto-saves to verification_list/** ✅

### Step 3: Submit to GitHub (EASY!)

**CHOOSE ONE METHOD:**

#### Method A - Automatic ⭐ RECOMMENDED

```bash
python auto_submit.py
```

✅ Finds your submission automatically  
✅ Adds to git, commits, and pushes  
✅ No git knowledge needed!  

#### Method B - Manual

```bash
git add verification_list/
git commit -m "Add Pi Value World submission for YOUR_USERNAME"
git push
```

### Step 4: Create Pull Request

1. Go to your fork on GitHub
2. Click "Contribute" → "Open Pull Request"
3. Title: `feat: add submission for YOUR_USERNAME`
4. Include your details
5. Create PR!

### Step 5: Get Certificate!

After PR merge (1-3 days):
- Website auto-verifies ✅
- Pi counter increases (+0.01)
- Download certificate at: https://pivalue.iths.online/search
