Submission workflow has changed

Submissions are now accepted through GitHub PR with verification data.

📋 Updated Submission Process

  1. Run python piclalculation.py to generate your results.
  2. Copy your Submission ID and Verification Code.
  3. Add the JSON file to verification_list (or equivalent repo path).
  4. Create a Pull Request describing your run. Once merged, the web system scans the list, verifies the code, and generates the certificate.

✅ The previous on-site submit form has been retired. Please use the PR-based flow only.

🔎 Use Search to verify your submission status and certificate.
