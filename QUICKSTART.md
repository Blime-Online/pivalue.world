# 🥧 Pi Value World - Quick Start

**Get your certificate in 3 simple steps!**

---

## Step 1: Fork & Clone

```bash
# 1. Fork this repo on GitHub (click Fork button)
# 2. Clone YOUR fork:
git clone https://github.com/YOUR_USERNAME/pivalue.world.git
cd pivalue.world
```

---

## Step 2: Run the Script

```bash
python piclalculation.py
```

**What happens:**
1. Enter your GitHub username
2. Choose time: 2, 5, or 10 minutes
3. Script calculates 22/7 repeatedly
4. **You get a JSON file** with your results

**Save these codes:**
- Verification Code (16 characters)
- Submission ID (12 characters)
- File name: `pi_result_YOUR_USERNAME.json`

---

## Step 3: Upload to GitHub

1. **Go to your fork on GitHub**
2. **Navigate to** `verification_list/` folder
3. **Click** "Add file" → "Create new file"
4. **Name it:** `pi_result_YOUR_USERNAME.json`
5. **Paste** the JSON data from your computer
6. **Click** "Commit changes"
7. **Create Pull Request**

---

## After Merge (Automatic)

✅ GitHub Actions runs automatically  
✅ Your JSON is synced to database  
✅ Visit https://pivalue.iths.online/search  
✅ Search your Submission ID  
✅ Download your certificate! 🎉  

⚠️ **If certificate doesn't appear after merge:**  
- Wait 5-10 minutes for GitHub Actions to complete  
- Check if maintainer set up Supabase secrets  
- Contact maintainer if still not showing

---

## What You Need

- Python 3.6+ ([Download](https://www.python.org/))
- GitHub account (free)
- 5-15 minutes

---

## Tips

- Close other programs for better performance
- Choose 10 minutes for more impressive certificate
- **Save your codes** - you need them for the PR!

python piclalculation.py
