# 📋 Verification List

**This is where your Pi Value World submissions go!**

---

## How to Submit

1. Run `python src/piclalculation.py` on your computer
2. The script creates `pi_result_username.json` in this folder
3. **Copy the JSON content** from that file
4. Go to your fork on GitHub
5. Navigate to `verification_list/` folder
6. Click "Add file" → "Create new file"
7. Name it: `pi_result_YOUR_USERNAME.json`
8. Paste the JSON content
9. Commit changes
10. Create Pull Request

---

## What's Inside

Each JSON file contains:
- GitHub username
- Verification Code (16 chars)
- Submission ID (12 chars)
- Time limit chosen
- Calculations performed
- Precision achieved

---

## After PR Merge

Website scans this folder and:
✅ Validates your submission
✅ Generates certificate
✅ Makes it available at https://pivalue.world

Search your ID to download certificate!
