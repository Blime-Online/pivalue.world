Rules & Guidelines

Official rules, code of conduct, and copyright information

📜 Official Challenge Rules

Eligibility

  • Open to anyone with a GitHub account
  • Must use your own computer hardware
  • One submission per time limit per user
  • Participants must agree to these rules

Calculation Requirements

  • Must use the official piclalculation.py script
  • Script must not be modified in any way
  • Must run for the full selected duration (2, 5, or 10 minutes)
  • Early termination is allowed but results may vary

Submission Guidelines

  • All submissions must be genuine calculations
  • Fake or manipulated results are prohibited
  • Automated scripts or bots are not allowed
  • Multiple accounts for bypassing limits is forbidden

Verification Process

  • All submissions undergo manual review
  • Maintainers reserve the right to reject any submission
  • Decision of maintainers is final
  • Cheating results in permanent disqualification

🤝 Code of Conduct

Our Pledge

We pledge to make participation in Pi Value World a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity, level of experience, nationality, personal appearance, race, religion, or sexual identity.

Expected Behavior

  • Be respectful and inclusive
  • Welcome newcomers and help them learn
  • Accept constructive criticism gracefully
  • Focus on what's best for the community
  • Show empathy towards others

Unacceptable Behavior

  • Trolling, insulting comments, or personal attacks
  • Public or private harassment
  • Publishing others' private information without permission
  • Other unethical or unprofessional conduct

🔒 Security Policy

Reporting Vulnerabilities

If you discover a security vulnerability, please report it privately to:

Email: harinand@iths.online

DO NOT create public GitHub issues for security vulnerabilities.

Data Collection

We collect minimal data:

  • GitHub username (public information)
  • Calculation results (performance metrics)
  • Submission timestamp

We do NOT collect emails, passwords, or payment information.

⚖️ Rule Enforcement

Consequences of Violations

  • First violation: Warning and submission removal
  • Second violation: Temporary ban from submissions
  • Third violation: Permanent ban and repository block

Appeals Process

If you believe you were unfairly penalized, you can appeal by emailing:

Email: harinand@iths.online

Include your submission details and explanation.

🔄 Changes to These Rules

We reserve the right to modify these rules at any time. Changes will be posted on this page with an updated revision date. Continued participation after changes constitutes acceptance of the new rules.
